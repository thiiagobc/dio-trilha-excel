# DIO • Santander — Excel com Inteligência Artificial (Trilha)

Este repositório reúne meus materiais, arquivos práticos e certificados do minicurso **“Excel com Inteligência Artificial”** da **DIO**, patrocinado pelo **Santander**.  
O conteúdo está organizado por módulos, com exercícios feitos em aula e entregáveis desenvolvidos durante a trilha.

## Estrutura do repositório

- **01 - Dominando_tabelas_com_excel**
  - Conteúdos e exercícios sobre **tabelas**, **tabela dinâmica**, **formatação condicional** e **validação de dados**
  - Planilha **Carteira.xlsx**: controle de investimentos (configurações, aporte mensal, patrimônio e dividendos)

- **02 - Trabalhando_com_formulas_no_excel**
  - Arquivos práticos aplicando **fórmulas e funções essenciais**
  - Exemplos com **datas/horas**, **buscas**, **texto**, **matemática** e casos de uso

- **03 - Criando_dashboards_com_excel**
  - Materiais e práticas para **criação de dashboards**
  - Organização e preparação de dados (incluindo conceitos trabalhados no módulo)
  - Arquivo **Dashboard.xlsx**: dashboard desenvolvido durante o módulo

## Principais aprendizados

- Estruturar dados de forma correta para análise (tabelas e organização)
- Automatizar tarefas com fórmulas e funções do Excel
- Aplicar validação de dados e regras de consistência para evitar erros
- Criar visualizações e dashboards para leitura rápida de indicadores
- Entender o fluxo de preparo de dados para análise (transformações e organização)
